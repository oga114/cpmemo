# cpmemo

## Overview
長文作成補助アプリ

## Requirement
cupertino_icons: ^1.0.2
shared_preferences: ^2.2.0

## Usage
長文を入力し右上のクリップボードボタンからコピーできます。あとは任意の場所にペーストして使います。

また右上に#はタップすると＃が入力されます。タグ入力を想定しています。

## Features
とにかくシンプル。
